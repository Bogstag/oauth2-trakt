This package provides Microsoft OAuth 2.0 support for the PHP League's [OAuth 2.0 Client](https://github.com/thephpleague/oauth2-client).

## Installation

To install, use composer:

```
composer require bogstag/oauth2-trakt
```




## Testing

``` bash
$ ./vendor/bin/phpunit
```
